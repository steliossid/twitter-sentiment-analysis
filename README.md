# Sentiment Analysis Tool for Twitter
Sentiment Analysis Tool for Twitter using Python 3.5 and MongoDB

## Requirements
* Twitter App
* Python 3.5
* MongoDB 3.4.5
* #### Python Libraries
  * tweepy
  * pymongo
  * nltk
  * textblob
  * matplotlib
